# 🇫🇷 French Learning Journal

Auto-generated daily from inbox notes via [GitHub Actions](https://github.com/BK3820/language-learning-repo/actions) + [GitHub Models](https://docs.github.com/en/github-models).

## 📖 Browse the data

- 🗣 [**Verbs**](data/verbs.md) — full conjugations (présent / passé composé / imparfait / futur simple)
- 📚 [**Vocabulary**](data/vocabulary.md) — nouns with gender, plural, English meaning
- 🔗 [**Connectors**](data/connectors.md) — discourse markers with example sentences
- 🎨 [**Adjectives**](data/adjectives.md) — masc / fem / plurals + meaning
- 📖 [**Daily Stories**](data/stories.md) — ~150-word French stories with vocabulary

## ➕ Add new words

The easiest way:

1. [**Open a new "Daily French words" issue →**](https://github.com/BK3820/language-learning-repo/issues/new?template=daily-words.yml)
2. Paste your verbs, vocabulary, connectors, adjectives into the form (one per line).
3. Submit. A bot will conjugate / translate them, commit to `data/`, and close the issue.

You can also push a Markdown file to `inbox/` directly — see the [README](https://github.com/BK3820/language-learning-repo#readme) for the format and the included ChatGPT prompt template.
